# TDD「Locker Robot」

1. given available locker 
    when save bag
    then return ticket
    
2. given locker full 
    when save bag
    then throw error
    
3. given valid ticket 
    when get bag
    then return bag
    
4. given invalid ticket 
    when get bag
    then throw error
    
5. given duplicate ticket 
    when get bag
    then throw error
