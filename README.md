# TDD「Locker Robot」

1. given a bag 
    when locker available 
    then return ticket
    
2. given a bag 
    when locker full 
    then throw error
    
3. given valid ticket 
    when press get button 
    then return bag
    
4. given invalid ticket 
    when press get button 
    then throw error
    
5. given duplicate ticket 
    when press get button 
    then throw error
